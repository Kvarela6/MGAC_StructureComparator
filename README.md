# MGAC_comparator_QEOut

A crystal structure comparison module developed for the MGAC Crystal Structure Prediction (CSP) workflow.  
This tool is designed to detect equivalent crystal structures and remove duplicates during large-scale evolutionary searches.

Although developed within the MGAC framework, the comparator is designed as a modular tool that can be integrated into other crystal structure prediction or structural analysis codes.

Currently, this version performs structure comparisons using Quantum ESPRESSO (QE) output files.  
A more general version of the project, based on CIF files, is currently under development.
